DiscTools
==========

About
-----
This library is about discord.py helper classes, functions and alternatives.

Installation
------------
``pip install DiscTools``
Or you can install the dev version by ``pip install git+https://github.com/WizzyBots/DiscTools.git@master#egg=DiscTools``
